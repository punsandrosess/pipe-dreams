Overview
Brand Identity

punsandrosess is a personal brand for AI-powered marketing tools, built in public. Every tool ships as a single HTML file — no login, no setup, no data leaving the user's machine. The visual identity reflects that philosophy: precise, editorial, and unapologetically opinionated.

Required — Tool Header
Logo & Tool Name
Every tool created under the punsandrosess brand must include the punsandrosess logo and the name of the tool in the header. This is non-negotiable. The logo anchors brand recognition; the tool name tells the user exactly where they are.

The header pattern for every tool follows this structure:

  • Line 1 — Logo: The punsandrosess wordmark, rendered in Playfair Display. "puns" in warm white (#F5F0E8), bold 900 weight. "androsess" in gold (#C9A84C), italic 700 weight. No space between them.
  • Line 2 — Context bar: A metadata line containing the category label (e.g., "Marketing"), fiscal year, tool description, and the tool name — separated by centered dots (·).

The tool name appears at the end of the context bar in Playfair Display at 13px, bold weight, with the second word in gold italic. For example:

punsandrosess
Marketing · FY2026 · Pipeline & Demand Generation · Mist Opportunities

Components
Card Pattern

All content sections use the card component: dark background (--card), subtle border, gold left accent (3px), and corner filigree marks in gold at top-right and bottom-left.

  • Padding: 18px 20px (desktop), 14px 12px (mobile)
  • Section labels: 9px Space Mono, gold, uppercase, .22em letter-spacing
  • The grid background uses 40px squares in faint gold on the page body
Components
Background Grid

The page body uses a subtle grid pattern created with CSS gradients:

background-image: linear-gradient(var(--gold-sub) 1px, transparent 1px), linear-gradient(90deg, var(--gold-sub) 1px, transparent 1px); background-size: 40px 40px;

Checklist
New Tool Requirements

Before shipping any new tool, verify the following:

  • Header: Includes the punsandrosess logo wordmark and the tool name in the context bar
  • Fonts: Loads Playfair Display, Space Mono, and Source Serif 4 from Google Fonts
  • Colors: Uses the standard CSS custom property palette defined in :root
  • Cards: Content sections use the card pattern with gold left border and corner filigrees
  • Grid: Page background uses the 40px gold grid pattern
  • Single file: Ships as one self-contained HTML file — no external dependencies beyond Google Fonts
  • No login: No authentication, no data transmission, everything runs client-side
  • Responsive: Works on mobile with appropriate breakpoints at 600px
